# VQ-VAE-Hopfield-
Relaxed version of the VQ-VAE based on modern Hopfield networks [1]

This work is an extension of the VQ-VAE paper [2] and the code provided here has been built upon [3].

[2]: https://arxiv.org/abs/1711.00937
